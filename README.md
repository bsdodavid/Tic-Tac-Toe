# Tic-Tac-Toe
Created a Advance Tic Tac Toe game using Html, css and javascript


##How to get started

To make a move, the player will use the mouse single click to mark a space. In this version, there is no provision to undo the move.And the first player always have the value X, As soon as the move is done, the game proceeds to the next playerâ€™s chance. As soon as the game over the remaining buttons remain dormant i.e. unclickable.
At each move, The game will show whose move (Player A or Player B) it is. When the game ends, it displays one of the following three outcomes:
Winner: Player A
Winner: Player B
